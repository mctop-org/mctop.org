explore

run mctop against a server to open the full-screen client. move through tools, resources, and prompts, fill a tool's arguments in a schema-driven form, run it, and read the result laid out as fields and tables, not raw json.

example

  mctop  ·  uvx mcp-server-time        ● connected
 TOOLS (2)              |  get_current_time
 ▌ get_current_time     |  get current time in a
   convert_time        |  specific timezone
 RESOURCES (0)         |  timezone* string
 PROMPTS (0)           |  enter to call

the result view reads the data: dates, yes/no, and grouped numbers are formatted, nested objects become sections, and arrays of records become selectable tables. press t for raw json, T for the protocol trace, y to copy.
