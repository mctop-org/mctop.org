script

skip the ui when you only need the answer. mctop ls lists what a server exposes; mctop call runs one tool and prints the structured result on stdout, ready to pipe into jq or a script.

example
$ mctop call "uvx mcp-server-time" get_current_time timezone=UTC
{
  "timezone": "UTC",
  "datetime": "2026-06-27T04:41:00+00:00",
  "day_of_week": "Saturday"
}

arguments are key=value pairs. values that look like json (numbers, booleans, arrays, objects) are typed; everything else is a string. pass a whole object with --json '{...}'. exit status reflects the call, so it composes in pipelines.
